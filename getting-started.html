2.3 Getting Started

Concourse is distributed as a single concourse binary, making it easy to run just about anywhere, especially with Docker.

$ wget https://concourse-ci.org/docker-compose.yml
$ docker-compose up
Creating docs_concourse-db_1 ... done
Creating docs_concourse_1    ... done

Concourse will be running at 127.0.0.1:8080.

Next, install fly by downloading it from the web UI and target your local Concourse:

$ fly -t tutorial login -c http://127.0.0.1:8080
logging in to team 'main'

target saved

Then, head to Hello World to kick the tires!
